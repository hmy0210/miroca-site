利用規約

最終更新日: 2026-03-29

第1条（定義）

本規約において、以下の用語は次の意味で使用します。

  1. 「本サービス」とは、BLST WEB（以下「当社」）が運営するウェブサイト「MIROCA（ミロカ）」（https://miroca.one）およびこれに付随するサービスをいいます。
  2. 「利用者」とは、本サービスを利用する全ての方をいいます。
  3. 「掲載業者」とは、本サービスにおいて情報が掲載されるハウスクリーニング等のサービス事業者をいいます。
  4. 「MIROCAスコア」とは、当社が独自に開発したアルゴリズムにより算出される評価指標をいいます。

第2条（サービスの性質）

  1. 本サービスは、公開されている情報を独自に収集・分析し、中立的な比較情報を提供するメタ検索エンジンです。
  2. 本サービスは特定の業者を推薦・紹介するものではありません。
  3. MIROCAスコアを含む全ての指標は、公開データの統計的分析に基づく当社の独自著作物であり、絶対的な評価ではありません。
  4. 業者との契約・取引は利用者と業者の間で直接行われるものであり、当社は当事者ではありません。

第3条（データの収集と分析）

  1. 本サービスは、以下の公開情報を収集・分析しています:
    • 各種口コミプラットフォームに公開されている評価・口コミ件数等の数値データ
    • Googleマップに公開されている事業者情報（評価、口コミ件数、所在地等）
    • 事業者が公開しているウェブサイト上の情報（料金、サービス内容等）
  2. 口コミの原文は本サービス上に公開しません。口コミテキストの統計的分析結果（キーワード出現率等の数値）のみを公開します。
  3. 分析結果は当社のアルゴリズムによる独自の著作物です。

第4条（免責事項）

  1. 本サービスに掲載される情報の正確性、完全性、最新性について、当社は保証しません。
  2. 本サービスの利用により生じた損害（業者との取引に関する損害を含む）について、当社は一切の責任を負いません。
  3. MIROCAスコアおよびAI分析コメントは、統計的分析に基づく参考情報であり、業者の品質を保証するものではありません。
  4. 本サービスに掲載される料金情報は、収集時点のデータに基づくものであり、実際の料金と異なる場合があります。必ず業者に直接ご確認ください。
  5. 本サービスの提供の中断、遅延、データの消失等により生じた損害について、当社は責任を負いません。

第5条（知的財産権）

  1. 本サービスに掲載される以下のコンテンツの著作権は当社に帰属します:
    • MIROCAスコアおよびその算出アルゴリズム
    • 口コミ解析指標（リピート率、時間厳守率、追加料金リスク等）
    • AI分析コメント
    • サイトデザイン、ロゴ、記事コンテンツ
  2. 当社の許諾なく、本サービスのコンテンツを複製・転載・スクレイピング・商業利用することを禁止します。

第6条（掲載業者に関する方針）

  1. 本サービスは、公開情報の分析に基づき業者情報を掲載しています。掲載にあたり、業者の許諾は要しません。
  2. 掲載情報に事実と異なる点がある場合、業者は当社に対し修正を請求できます。お問い合わせよりご連絡ください。
  3. 業者からの掲載停止の要請については、掲載方針に基づき対応します。

第7条（禁止行為）

利用者は、以下の行為を行ってはなりません。

  1. 本サービスのデータを自動収集（スクレイピング）する行為
  2. 本サービスの運営を妨害する行為
  3. 本サービスのコンテンツを無断で複製・転載する行為
  4. 虚偽の情報を用いた問い合わせ・クレーム

第8条（規約の変更）

当社は、必要に応じて本規約を変更できるものとします。変更後の規約は本ページに掲載した時点で効力を生じます。

第9条（準拠法・管轄）

本規約は日本法に準拠し、本サービスに関する紛争は横浜地方裁判所を第一審の専属的合意管轄裁判所とします。
